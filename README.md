## Crear un git
### Para agregar y hacer seguimiento de arvhivos
```bash
git add .
```
### Para hacer commit
```bash
git commit -m "Primer commit"
```

### Estado de mis archivos


```bash
git status
```

### Historial de cambios


```bash
git log
```

### cambiar el nombre de un branch


```bash
git branch -M main
```
